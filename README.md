# Otto's Notes
Personal notes written in typst, built with shiroa. Check out the [intro.typ](./intro.typ) for more information about me.

The website is supposed to be live at [https://docs.OttoQwQ.me](https://docs.OttoQwQ.me), and the source code is available at [https://othodeng.github.io](https://othodeng.github.io).

## Usage
The source files are written in typst, and the output files are generated by shiroa. 

Visit [shiroa]("https://github.com/Myriad-Dreamin/shiroa") for instructions on installation.

To serve the notes locally, run the following command in the root directory of the repository:
```bash
shiroa serve
```

To build the notes, run the following command in the root directory of the repository:
```bash
shiroa build
```
You may costumize the output directory in `book.typ`.
